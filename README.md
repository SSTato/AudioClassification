# AudioClassification
Identify and classify the sound generated by the coal conveying belt, and alarm the abnormal sound

### 项目用途
使用输煤带产生的音频文件生成梅尔频谱特征图，通过特征图进行训练，实现声音分类，同时可通过设置阈值，分辨未知声音，实现异常声音报警。
